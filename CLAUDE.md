# PDFラベルメーカー

## プロジェクト概要
テキストを入力し、ラベル用紙に合わせたPDFを生成するWebアプリケーション

##　技術仕様
- HTML + JavaScript(サーバー不要、ブラウザだけで動く)
- PDF生成にはjsPDFライブラリを使う(CDNから読み込む)

## ラベルの仕様
- 用紙サイズ: A4(210mm * 297mm)
- 余白(上下左右)を設定できる
- ラベルのサイズ(幅 * 高さ)を設定できる
- ラベル間の余白を設定できる
- 用紙プリセットをドロップダウンから選択すると、余白・ラベルサイズ・ラベル間余白を自動入力する(選択後も手動で微調整可能)
  - カスタム(手動設定、初期値)
  - A4 72面(ラベル25.4mm×10mm、余白:上下10.2mm、左右7.6mm、間隔:横2.54mm 縦0mm)
  - A4 24面(ラベル64.3mm×33.9mm、余白:上13.5mm 下12.6mm 左右6.8mm、間隔0mm)
  - A4 44面(ラベル48.3mm×25.4mm、余白:上下10.6mm 左右6.4mm、間隔0mm)
- テキストを入力し、すべてのラベルに同じ内容を印字する
- 余白・ラベルサイズ・ラベル間余白から、用紙に配置できる行数・列数を自動計算する
- ラベル数が用紙1ページに収まりきらない場合は、自動的に複数ページに分割してPDFを生成する
- ラベルには切り取り線の目安として枠線(罫線)を表示する
- PDF生成前に、画面上でラベル配置のプレビューを表示する
- 入力した設定(余白・ラベルサイズなど)はブラウザのlocalStorageに保存し、次回開いたときに自動復元する

## テキストの表示ルール
- ラベル内のテキストは左上揃えで配置する
- テキスト量やラベルサイズに応じて、フォントサイズを自動的に縮小し枠内に収める

## 日本語フォント対応
- jsPDFは標準で日本語(非ASCII)フォントを埋め込まないため、そのままでは文字化けする
- Webフォント(例: Noto Sans JP)を取得してBase64化し、jsPDFに埋め込んで日本語を印字する

## 入力値のバリデーション
- 余白やラベルサイズが不正な値(負数、0、用紙サイズを超える値など)の場合は、エラーメッセージを表示しPDF生成を行わない
- ラベル間の余白はラベルを隙間なく配置できるよう0を許可する(負数のみ不正値とする)

## コーディング規約
- コメントは日本語で書く
- 1つのHTMLファイルにまとめる(外部CSSや外部JSファイルは作らない)
